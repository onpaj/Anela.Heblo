# Type-Safe ArticleStatus in Response DTOs Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Align `GetArticleResponse.Status` and `ArticleListItemDto.Status` on the `ArticleStatus` enum (matching the already-correct `GenerateArticleResponse`), regenerate the TypeScript client so the generated types are typed instead of `string`, and remove the two unsafe `as ArticleStatus` casts from the frontend hooks.

**Architecture:** Pure type-system refactor inside the Article vertical slice. The global `JsonStringEnumConverter` registered in `backend/src/Anela.Heblo.API/Program.cs:121` already serializes `ArticleStatus` to the same string wire format (`"Generated"`, `"Queued"`, ...), so swapping the C# property type from `string` to `ArticleStatus` produces a byte-identical JSON body. NSwag then emits the TypeScript field as `ArticleStatus | undefined`, and the frontend hook drops its `as ArticleStatus` cast while keeping the `?? ArticleStatus.Queued` fallback because the generated property remains optional. A focused serialization wire-format test locks the contract.

**Tech Stack:** .NET 8, xUnit + FluentAssertions + Moq, MediatR, ASP.NET Core MVC, `System.Text.Json` with `JsonStringEnumConverter`, NSwag (OpenAPI → TS client), React 18 + TanStack Query, TypeScript.

## Task 3: Backend — Wire-format snapshot tests (FR-5)

A focused unit test that serializes each DTO with the same `JsonSerializerOptions` shape the API uses (`new JsonStringEnumConverter()` in `Converters`) and asserts the substring `"status":"<EnumName>"` is present. This locks the JSON contract so a future regression in either the converter registration or the property type is caught at test time. Pattern reference: `backend/test/Anela.Heblo.Tests/Features/Smartsupp/WebhookAudit/SmartsuppWebhookAuditControllerTests.cs:15-19`.

We cover two enum members per DTO: `Generated` and `Queued`. `Queued` (the zero-valued enum member) ensures the default for an uninitialized instance still serializes as the expected string, guarding against accidental switches to numeric serialization.

**Files:**
- Create: `backend/test/Anela.Heblo.Tests/Article/UseCases/ArticleStatusWireFormatTests.cs`

- [ ] **Step 1: Write the failing test**

Create `backend/test/Anela.Heblo.Tests/Article/UseCases/ArticleStatusWireFormatTests.cs` with:

```csharp
using System.Text.Json;
using System.Text.Json.Serialization;
using Anela.Heblo.Application.Features.Article.UseCases.GetArticle;
using Anela.Heblo.Application.Features.Article.UseCases.ListArticles;
using Anela.Heblo.Domain.Features.Article;
using FluentAssertions;

namespace Anela.Heblo.Tests.Article.UseCases;

public class ArticleStatusWireFormatTests
{
    private static readonly JsonSerializerOptions ApiJsonOptions = new()
    {
        Converters = { new JsonStringEnumConverter() },
    };

    [Theory]
    [InlineData(ArticleStatus.Generated, "\"status\":\"Generated\"")]
    [InlineData(ArticleStatus.Queued, "\"status\":\"Queued\"")]
    public void GetArticleResponse_SerializesStatusAsEnumName(ArticleStatus status, string expectedFragment)
    {
        var response = new GetArticleResponse { Status = status };

        var json = JsonSerializer.Serialize(response, ApiJsonOptions);

        json.Should().Contain(expectedFragment);
    }

    [Theory]
    [InlineData(ArticleStatus.Generated, "\"status\":\"Generated\"")]
    [InlineData(ArticleStatus.Queued, "\"status\":\"Queued\"")]
    public void ArticleListItemDto_SerializesStatusAsEnumName(ArticleStatus status, string expectedFragment)
    {
        var item = new ArticleListItemDto { Status = status };

        var json = JsonSerializer.Serialize(item, ApiJsonOptions);

        json.Should().Contain(expectedFragment);
    }
}
```
