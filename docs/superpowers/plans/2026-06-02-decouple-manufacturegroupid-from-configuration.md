# Decouple ManufactureGroupId from Configuration Module — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Move `ManufactureGroupId` off the cross-cutting `/api/configuration` response into a new Manufacture-owned anonymous endpoint `/api/manufacture/settings`, replacing the raw `"ManufactureGroupId"` config string literal with a typed constant scoped to the Manufacture module.

**Architecture:** New `ManufactureSettingsController` (separate from `[Authorize]`-decorated `ManufactureOrderController`) exposes `GET /api/manufacture/settings` as anonymous. A new MediatR use case (`UseCases/GetManufactureSettings/`) reads `IConfiguration["ManufactureGroupId"]` via a `ManufactureConfigurationKeys.GroupId` constant. Frontend gets a new `useManufactureSettingsQuery()` hook mirroring `useConfiguration.ts`; three call sites swap which hook they read. The old field is removed from `GetConfigurationResponse` and its handler in the same release — single Docker image deploys both sides together.

**Tech Stack:** .NET 8 + MediatR + ASP.NET Core MVC + xUnit + FluentAssertions + Moq (backend); React + TanStack Query + NSwag-generated TypeScript client (frontend).

```csharp
using MediatR;
using Microsoft.Extensions.Configuration;
using Microsoft.Extensions.Logging;

namespace Anela.Heblo.Application.Features.Manufacture.UseCases.GetManufactureSettings;

/// <summary>
/// Handler for the anonymous Manufacture bootstrap-settings endpoint.
/// Reads non-secret identifiers from IConfiguration; returns null when a key is missing or empty.
/// </summary>
public class GetManufactureSettingsHandler
    : IRequestHandler<GetManufactureSettingsRequest, GetManufactureSettingsResponse>
{
    private readonly IConfiguration _configuration;
    private readonly ILogger<GetManufactureSettingsHandler> _logger;

    public GetManufactureSettingsHandler(
        IConfiguration configuration,
        ILogger<GetManufactureSettingsHandler> logger)
    {
        _configuration = configuration ?? throw new ArgumentNullException(nameof(configuration));
        _logger = logger ?? throw new ArgumentNullException(nameof(logger));
    }

    public Task<GetManufactureSettingsResponse> Handle(
        GetManufactureSettingsRequest request,
        CancellationToken cancellationToken)
    {
        var groupId = _configuration[ManufactureConfigurationKeys.GroupId];
        if (string.IsNullOrEmpty(groupId))
        {
            groupId = null;
        }

        _logger.LogDebug("GetManufactureSettings resolved ManufactureGroupId hasValue={HasValue}", groupId is not null);

        return Task.FromResult(new GetManufactureSettingsResponse
        {
            ManufactureGroupId = groupId
        });
    }
}
```

## Task 7: Implement `ManufactureSettingsController` (GREEN)

**Files:**
- Create: `backend/src/Anela.Heblo.API/Controllers/ManufactureSettingsController.cs`

- [ ] **Step 1: Create the controller**

> **Route is literal.** Use `[Route("api/manufacture/settings")]` — not `[Route("api/[controller]")]` which would resolve to `/api/manufacturesettings` and break the contract.
>
> **`[AllowAnonymous]` is explicit.** The codebase's pattern is "anonymous-by-omission of `[Authorize]`", but a future global fallback policy could break the SPA bootstrap. Explicit attribution documents intent and is defensive.

```csharp
using Anela.Heblo.Application.Features.Manufacture.UseCases.GetManufactureSettings;
using MediatR;
using Microsoft.AspNetCore.Authorization;
using Microsoft.AspNetCore.Mvc;

namespace Anela.Heblo.API.Controllers;

/// <summary>
/// Anonymous endpoint exposing Manufacture-module bootstrap settings required by the SPA
/// before authentication completes (e.g. the Entra group ID used for responsible-person lookups).
/// Separated from ManufactureOrderController (which is [Authorize]) so the security posture
/// stays unambiguous.
/// </summary>
[ApiController]
[Route("api/manufacture/settings")]
public class ManufactureSettingsController : ControllerBase
{
    private readonly IMediator _mediator;

    public ManufactureSettingsController(IMediator mediator)
    {
        _mediator = mediator;
    }

    [HttpGet]
    [AllowAnonymous]
    public Task<GetManufactureSettingsResponse> GetSettings(CancellationToken cancellationToken)
        => _mediator.Send(new GetManufactureSettingsRequest(), cancellationToken);
}
```
