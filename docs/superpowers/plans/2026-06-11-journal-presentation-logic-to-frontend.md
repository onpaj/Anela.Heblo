# Move Journal Search Presentation Logic to Frontend Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Move content preview truncation and search-term extraction off the backend `SearchJournalEntriesHandler` and into the frontend, consolidating two divergent truncation implementations (200 vs 150 chars) into a single client-side helper used by both browse and search views and by both `JournalList` and `JournalTab`.

**Architecture:** Backend stops shaping data for display: `CreateContentPreview` / `ExtractHighlightTerms` are deleted, `SearchJournalEntryDto` loses `ContentPreview` and `HighlightedTerms` and gains raw `Content`, and `JournalEntryMapper.ToSearchDto` now populates that `Content` directly from the domain entity. Frontend gets a new `journalPreview.ts` module (one file, no React) exporting `MAX_PREVIEW_LENGTH = 200` and a `truncateContent(content, { searchQuery? })` helper that centers a window on the first case-insensitive match when a query is present, or head-truncates otherwise. Both `JournalList.tsx` (browse + search rows) and `JournalTab.tsx` (catalog product detail) call that helper. Highlighting is dropped entirely — the field was already populated, transported, and unused.

**Tech Stack:** .NET 8 (ASP.NET Core, MediatR, FluentAssertions, Moq, xUnit) backend; React + TypeScript frontend (Jest + React Testing Library, OpenAPI/NSwag-generated TypeScript client regenerated on backend build).

Create `frontend/src/components/pages/Journal/journalPreview.ts` with:

```typescript
export const MAX_PREVIEW_LENGTH = 200;

const ELLIPSIS = "...";

export interface TruncateOptions {
  searchQuery?: string;
  maxLength?: number;
}

export function truncateContent(
  content: string,
  options?: TruncateOptions,
): string {
  if (!content) {
    return "";
  }

  const maxLength = options?.maxLength ?? MAX_PREVIEW_LENGTH;
  const searchQuery = options?.searchQuery?.trim() ?? "";

  if (searchQuery.length === 0) {
    return headTruncate(content, maxLength);
  }

  const matchIndex = content
    .toLowerCase()
    .indexOf(searchQuery.toLowerCase());

  if (matchIndex === -1) {
    return headTruncate(content, maxLength);
  }

  const start = Math.max(0, matchIndex - Math.floor(maxLength / 2));
  const length = Math.min(maxLength, content.length - start);
  let preview = content.substring(start, start + length);

  if (start > 0) {
    preview = ELLIPSIS + preview;
  }
  if (start + length < content.length) {
    preview = preview + ELLIPSIS;
  }

  return preview;
}

function headTruncate(content: string, maxLength: number): string {
  if (content.length <= maxLength) {
    return content;
  }
  return content.substring(0, maxLength) + ELLIPSIS;
}
```
